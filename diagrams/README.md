# Diagrams

Source files for all diagrams used in the book.

## Formats

- **Mermaid** (`.mmd`) — For flowcharts, sequence diagrams, architecture diagrams
- **draw.io** (`.drawio`) — For complex visual diagrams
- **PlantUML** (`.puml`) — For UML diagrams

## Directory Structure

Place source files in `src/` with names matching their chapter and figure number:

```
src/
├── ch01-fig01-sdlc-phases.mmd
├── ch04-fig01-ai-tool-taxonomy.mmd
└── ch07-fig01-agent-workflow.drawio
```

Exported images go into the corresponding chapter's `figures/` directory.
