# Prompt Library

Curated, ready-to-use prompts organized by SDLC phase.

## Directory Structure

| Directory | SDLC Phase | Chapters |
|-----------|-----------|----------|
| `requirements/` | Requirements & Planning | Ch 5 |
| `design/` | Design & Architecture | Ch 6 |
| `coding/` | Implementation | Ch 7, 8 |
| `testing/` | Testing & QA | Ch 9 |
| `devops/` | CI/CD & Operations | Ch 10, 11 |

## How to Use

Each directory contains Markdown files with prompts categorized by task type. Prompts include:

- **Context placeholders** — marked with `[brackets]` for you to fill in
- **Expected output format** — what the AI should return
- **Variations** — alternative phrasings for different tools
