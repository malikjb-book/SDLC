# Appendix A: Glossary

| Term | Definition |
|------|-----------|
| **Agentic AI** | AI systems capable of autonomous multi-step reasoning and action |
| **AI Copilot** | An AI assistant that works alongside a developer in an IDE |
| **CI/CD** | Continuous Integration / Continuous Delivery |
| **Coding Agent** | An autonomous AI that can write, test, and refactor code |
| **DORA Metrics** | DevOps Research and Assessment metrics (deployment frequency, lead time, MTTR, change failure rate) |
| **Foundation Model** | A large pre-trained model (e.g., GPT, Claude) that can be adapted for various tasks |
| **IaC** | Infrastructure as Code |
| **LLM** | Large Language Model |
| **MCP** | Model Context Protocol — a standard for tool integration with AI models |
| **SDLC** | Software Development Lifecycle |
| **Shift-Left** | Moving activities like testing and security earlier in the SDLC |
